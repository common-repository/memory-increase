Attempts to increase the memory limit to 32 MB on WordPress versions less than WordPress 2.5. '''Do not use in WordPress 2.5!'''

== Description ==

'''WordPress 2.5''' fixes this issue, so this plugin shouldn't be used with WordPress 2.5. It can be used with WordPress 2.3.x and below to fix the issue in previous WordPress versions.

== Installation ==

1. Upload the 'dragonu_memory_increase' folder to the 'wp-content\plugins' directory.
1. Activate the plugin titled "DragonU Memory Increase" through the "Plugins" menu in WordPress.
